# Hangman Game

A modern, React-based Hangman game that allows users to play with randomly generated words or input their own custom words.

## Features

- Random word generation via API
- User-defined word input
- Animated hangman stick figure
- On-screen keyboard input display
- Win/loss messaging
- Game reset functionality
- Session storage for persistent game state

## Getting Started

### Play Online

Try the game instantly: [Hangman Game Live](https://simonprogai.github.io/hangman_app/)

### Local Development

1. **Clone the repository:**
   ```sh
   git clone https://github.com/SimonProgAI/hangman_app.git
   cd hangman_app
   ```
2. **Install dependencies:**
   ```sh
   npm install
   ```
3. **Start the development server:**
   ```sh
   npm start
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.
